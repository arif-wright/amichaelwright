# Site Images

Drop final site images in this folder.

Current files:

- `Book 1.png` is used as the live book cover on the site.

Recommended future filenames:

- `the-shattered-pact-cover.jpg` for the book cover
- `hero-background.jpg` for the hero background
- `og-image.jpg` for social sharing previews

After changing filenames, update the image constants at the top of `app/page.tsx`.
